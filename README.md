# Class Management System

Repository layout:

- `frontend/` React app (current working application)
- `backend/` Backend service folder (reserved for API/server code)

## Frontend

```bash
cd frontend
npm install
npm start
```

Build:

```bash
cd frontend
npm run build
```

Test:

```bash
cd frontend
npm test -- --watchAll=false
```

## Backend

`backend/` is ready for server implementation (for example Node.js + Express).
